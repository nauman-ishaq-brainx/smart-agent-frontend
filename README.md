# AI Assistant Frontend

This is the React-based frontend for the **AI Assistant App**, which allows users to interact with a smart assistant that can:

- Answer questions from uploaded PDF documents
- Send professional emails
- Schedule and retrieve Google Calendar events

---

##  Features

-  PDF-based question answering using vector search
-  Email sending via natural language prompts
-  Calendar event scheduling with attendee support
-  Modular architecture with centralized API service
- ⚛ Built using React, Axios, and Bootstrap

---

## Environment Setup
Create a .env file inside the ai-assistant-app/ directory:

REACT_APP_API_BASE_URL=http://localhost:5000/api/v1
